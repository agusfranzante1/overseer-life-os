🗄 Overseer — Backup local

Para que esto funcione, tenés que abrir este HTML en el mismo navegador donde tenés Overseer corriendo en localhost:3001. Como los localStorage son por origen, este HTML necesita levantarse desde el mismo origen.

Cómo correrlo: en una terminal en la raíz del proyecto, hacé:
npx http-server scripts -p 4000 -c-1 y abrí http://localhost:4000/export-local-data.html.
O directamente pegá el script de abajo en la consola de DevTools en localhost:3001.

Detecta todos los keys overseer-* en tu localStorage y los descarga como un JSON.

Comando alternativo para consola DevTools

(() => {
  const out = {};
  for (let i = 0; i < localStorage.length; i++) {
    const k = localStorage.key(i);
    if (k && k.startsWith('overseer-')) {
      try { out[k] = JSON.parse(localStorage.getItem(k)); }
      catch { out[k] = localStorage.getItem(k); }
    }
  }
  const payload = { exportedAt: new Date().toISOString(), version: 1, data: out };
  const blob = new Blob([JSON.stringify(payload, null, 2)], { type: 'application/json' });
  const url = URL.createObjectURL(blob);
  const a = document.createElement('a');
  a.href = url;
  a.download = `overseer-backup-${new Date().toISOString().slice(0,10)}.json`;
  a.click();
  URL.revokeObjectURL(url);
  console.log('Exported', Object.keys(out).length, 'stores');
})();
